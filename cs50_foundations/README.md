## 📦 Repository Structure

cs50_foundations/
├─ pset1/          # C basics (loops, conditionals, functions)

├─ pset3/          # Sorting and searching algorithms

Each `psetX/` contains:

- A local `README.md` with explanation and usage
- `Makefile` when applicable
